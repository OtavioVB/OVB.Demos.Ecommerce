# OVB.Demos.Ecommerce

Esse repositório é caracterizado por manter o projeto de desenvolvimento de API para Ecommerce Online utilizando a plataforma de desenvolvimento .NET.

## Conceitos Aplicados

- CQRS
- Consistência Eventual
- Mensageria
- SOLID
- Injeção de Dependência
- Notification Pattern
- Adapter Pattern
- Factory Pattern
- Builder Pattern
- Singleton Pattern
- Command Pattern
- Unit Of Work Pattern
- Repository Pattern
